# GPS Speedometer・Speed Tracker - Features

This page describes the main features of the **GPS Speedometer・Speed Tracker** app.

## Main features

### 1. Real-time GPS speed tracking
The app shows your current speed live using GPS data from your iPhone.

### 2. Digital speedometer
A clean, modern interface with large numbers for easy reading.

### 3. Analog speedometer
A classic dial-style view for users who prefer a traditional speedometer look.

### 4. HUD mode
The display can be mirrored to help project speed information onto a windshield during night driving.

### 5. Trip route map
The app records the route of your trip and displays it on a map.

### 6. Detailed trip statistics
Users can review:

- current speed
- maximum speed
- average speed
- trip distance
- travel history

### 7. Unlimited trip history
Trip data can be stored and reviewed later.

### 8. MPH and KMH switching
The app supports both miles per hour and kilometers per hour.

### 9. Simple Mode
A distraction-free display with very large speed numbers.

### 10. Premium upgrade
The premium version removes ads and unlocks additional functionality.

## Common use cases

GPS Speedometer・Speed Tracker can be used for:

- car driving
- cycling
- motorcycle riding
- delivery work
- travel tracking
- road trips
- checking speed in areas where a dashboard is hard to read
